# 广州地铁计价器

### 项目介绍
这是一个广州地铁计价器(线路更新中...)  
[项目预览](https://hymanchoi.github.io/Guangzhou-Metro-Meter/dist/index.html)

### 项目技术
Vue + ElementUI + Dijkstra算法

### 安装教程
1. 安装依赖
```
npm install/cnpm install
```
2. 运行项目
```
npm run serve
```
3. 压缩打包
```
npm run build
```

### 使用说明
1. 选择起点位置(起点线路 + 起点站点)
2. 选择终点位置(终点线路 + 终点站点)
3. 选择付费方式
4. 填写乘坐次数
5. 点击查看按钮
